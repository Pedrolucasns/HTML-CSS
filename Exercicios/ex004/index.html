HTML (HyperText Markup Language) é a linguagem padrão utilizada para estruturar e exibir conteúdo na web. Ele funciona como a base de qualquer página da internet, permitindo organizar textos, imagens, vídeos, links e outros elementos em um formato compreensível para os navegadores. Através de suas tags, que são comandos delimitados por < >, o HTML indica como cada parte do conteúdo deve ser exibida, como títulos, parágrafos, listas ou tabelas. Apesar de não ser uma linguagem de programação, é essencial para a criação de websites, sendo frequentemente usada em conjunto com CSS e JavaScript para adicionar estilos e interatividade.
