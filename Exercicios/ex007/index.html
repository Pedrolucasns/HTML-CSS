Imagens Dinâmicas

Para fazer imagens dinâmicas, vamos precisar de uma imagem em 3 tamanhos diferentes, por exemplo: 300x300px; 700x700px e 1000x1000px.

As tags usadas serão "picture" e "img". Por padrão, a foto maior será carregada, mas para ajustar os tamanhos, usaremos os atributos da tag source, como type, srcset e media.
